# 秦皇岛防贫工作系统

> 为扶贫事业做点贡献吧

## 开始项目

```bash
# 克隆
git clone https://gitee.com/xhute/fupin.git

# 初始化
cd fupin
npm install
```


## 平时调试
```bash
npm run dev
```

## 生成并上传

```bash
npm run build
# 1、构建至dist目录
# 2、将dist上传至45.233.70.235服务器指定目录
```
